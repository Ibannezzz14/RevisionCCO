# Parcours Securite 231

Ce dossier prepare le futur parcours de revision Securite 231.

Structure:

- `data/course.json`: metadata et liste des modules a connecter plus tard.
- `modules/`: fiches de revision securite.
- `exercices/`: cas pratiques, scenarios et corrections.
- `assets/`: images, schemas, captures ou supports visuels.

Quand le contenu sera pret, le parcours pourra etre branche dans l'accueil deja existant.
